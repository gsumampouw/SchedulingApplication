How to run: If you start the application and you will see the login page. Type in the username and password to go to the home page. 
From the home page you can add, update or delete a customer. You must update a customer in order to add, update or delete an appointment. 
From the home page you can navigate to three different reports. The "Report: User Schedule" button will take you to a report that shows 
a list of appointments depending on the user selected. This report satisfies section A3f of the project.
